# AI_system - 栄養バランス判別分類器
-このプロジェクトは，機械学習を用いて「食事が健康的か不健康か」を栄養データから判別する分類器を構築した成果物です。 
-大学の講義 知的情報システム開発2（実践的システム開発演習）の一環として企画・実装しました。 
本講義は何かしら自主的に問題発見をしてそれに対して機械学習を用いて成果物を作ることが目的  
この講義は2025/4 ~ 2025/7 までの内容です  

##プロジェクトの目的  
健康的な食生活の啓発と栄養分析ツールの基礎構築   
栄養バランスの良い献立を考えることを目標に機械学習を用いて健康不健康を判断する分類器作成を目指した。 　
データ不足等の観点から健康不健康の分類が困難であったため「食べようとしている食事が不健康かどうか判断したい」という考えのもと”材料単位”で栄養素データを取得し学習させる方法を用いた。  

-日々の食事を栄養素ベースでモデル化し、機械学習で健康度を判定したい  
-データの前処理,特徴量選択,モデル評価まで一連のMLワークフローを実装する

### 学習  
使用モデル  
・Randomforest   
・lightGBM  
どちらも決定木を用いた機械学習の分類器  
モデルの違い  
randomforest:  
バギングによって複数の決定木から予測結果を平均化または多数決で結合する(並列処理)過学習に強いモデル  

lightgbm:  
ブースティングにより複数の弱い決定木を構築して前のモデルの間違えた部分に重みを置いて次モデルを改善する(直列処理)バイアスを減らすことができるモデル  

lightgbmは精度向上に向いているが計算負荷が高い、randomforestは安定性が高く比較的簡単であるが精度は劣る  

それぞれのモデルを使用して特徴量重要度によって特徴量を変更することによりモデルの性能を比較した
